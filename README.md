# Soumission Radar

This repository contains the public static deployment of [Soumission Radar](https://ramlaoui.github.io/soumission-radar/), a local-first tender submission readiness checker.

The tool runs in the browser: selected tender files are classified and packaged locally and are not uploaded by the site. Results still require human comparison with the original tender document.

Soumission Radar is independent and is not affiliated with ARCOP, DCMP, a contracting authority, or an official procurement portal.
