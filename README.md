# Soumission Radar

This repository contains the public static deployment of [Soumission Radar](https://ramlaoui.github.io/soumission-radar/), a French-first tender execution workspace.

The tool reads text PDFs and DOCX files in the browser, extracts cited requirements, links them to company files, answers grounded questions, and exports a final review package. Files are not uploaded by the public site. Results still require human comparison with the original DAO and its addenda.

Soumission Radar is independent and is not affiliated with ARCOP, DCMP, a contracting authority, or an official procurement portal.

The deployment vendors JSZip, Mammoth, and PDF.js. Their licenses are included under `vendor/`. No license is granted for the remaining code or branding.
